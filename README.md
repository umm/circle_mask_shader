# circle_mask_shader

Circle Mask Shader is mask utility for uGUI Image/RawImage.

## Requirement

Development on
- Unity 2019.3.11f ~

## Install

```shell
yarn add "umm/circle_mask_shader#^1.0.0"
```

## Usage

1. Create material and change shader to `UI/CircleMask`
2. Attach the material to Image/RawImage GameObject.
